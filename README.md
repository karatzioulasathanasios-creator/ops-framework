# OPS Framework - Operational Projective Stability

Code repository for "AT2020mot: exceptional polarization revealed by a cross-domain projective structure analysis"

## Overview
Implementation of the Operational Projective Stability (OPS) framework for detecting projective common-mode structure in multi-channel time series.

## Analysis Domains
1. Earth rotation parameters (IERS EOP C04)
2. Pulsar timing arrays (NANOGrav 15-year)
3. Tidal disruption event polarization (Floris et al. 2024)

## Requirements
- Python 3.8+
- NumPy, SciPy, Matplotlib
- Jupyter notebooks for reproducibility
